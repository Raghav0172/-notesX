# NotesX - University Notes Sharing Platform\n\nFull-stack project built with React.js (frontend), Node.js + Express.js (backend), MongoDB (database), Firebase Auth + Storage.\n\n## Structure\n- `backend/` Express API and Firebase/DB integration\n- `frontend/` React + Vite app\n\n## Setup\n1. `cd backend && npm install`\n2. copy `.env.template` to `.env` and configure Mongo + Firebase service account data.\n3. `npm run dev`\n4. `cd frontend && npm install`\n5. copy `.env.template` to `.env` and configure Firebase client settings.\n6. `npm run dev`\n\n## Notes\n- Backend uses Firebase admin SDK to verify bearer tokens and upload note files to Firebase Storage.\n- Upload flow requires `notesxToken` in browser localStorage from Firebase Auth user session.\n- Admin approval gate: new notes are saved as `approved=false` and only appear in list when approved.\n\n## Components\n- `backend/routes/notes.js`: core CRUD operations, search/filter, download counter, approval endpoint.\n- `frontend/src/pages/NoteList.jsx`: frontpage with search and list + direct download link.\n- `frontend/src/pages/UploadNote.jsx`: form for note upload (requires auth token).\n\n## Bring it to life\n- Spin up MongoDB Atlas / local Mongo and Firebase project.\n- Run backend and frontend as above.\n- Use Firebase Auth to obtain ID token and store as `localStorage.setItem('notesxToken', token)` while testing.\n

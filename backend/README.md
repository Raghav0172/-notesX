# NotesX Backend\n\nNode.js + Express.js API for the NotesX University Notes Sharing Platform.\n\n## Setup\n1. `cd backend`\n2. `npm install`\n3. Copy `.env.template` to `.env` and set values for `MONGODB_URI`, `FIREBASE_PROJECT_ID`, `FIREBASE_CLIENT_EMAIL`, `FIREBASE_PRIVATE_KEY`, `FIREBASE_STORAGE_BUCKET`.\n4. `npm run dev`\n\n## API endpoints\n- `GET /api/health`\n- `GET /api/notes` (filter by query `department, year, subject, search, sort=latest|popular`)\n- `GET /api/notes/:id`\n- `POST /api/notes` (authenticated, fields `title, department, year, subject, topic, description, tags`, file field `file`)\n- `PUT /api/notes/:id/approve` (authenticated admin)\n- `PUT /api/notes/:id/download`\n\n## Notes\n- Uses Firebase Authentication token (Bearer) on protected routes.\n- Requires deployed Firebase Storage for note files.\n- Notes default to `approved=false` until admin toggles approval.\n
